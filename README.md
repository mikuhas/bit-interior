# Bit Interior

Bit Interior は、React、TypeScript、Vite を使用して構築された、アイソメトリックな部屋の間取りデザインツールです。家具、ドア、窓の配置や、部屋の見た目のカスタマイズ、ブループリントの可視化が可能です。

## 技術スタック

- **フレームワーク:** React 19 (TypeScript)
- **ビルドツール:** Vite
- **データ管理:** TanStack Query (React Query)
- **スタイリング:** Vanilla CSS
- **ユーティリティ:** HTML5 Canvas (アイソメトリックおよびトップダウンレンダリング)

## アーキテクチャと原則

本プロジェクトは、`GEMINI.md` で定義された以下の厳格なアーキテクチャ原則に従っています。

1.  **宣言的なデータ管理:**
    - `useEffect` による手動フェッチは禁止。
    - すべての非同期操作（永続化、JSON インポートなど）は、**TanStack Query** (Queries/Mutations) を介して抽象化されます。
2.  **パフォーマンスとレンダリング:**
    - コンポーネントは最小単位に分割し、不要な再レンダリングを抑制。
    - ロジックはカスタムフック (`use...`) にカプセル化。
    - レンダリングロジックは UI コンポーネントから分離し、専用のユーティリティモジュールで管理することでコンポーネントをクリーンに保つ。
3.  **厳格な型定義:**
    - すべての定義で TypeScript を使用し、`any` 型の使用は厳格に禁止。

## プロジェクト構造

```text
src/
├── components/       # UI コンポーネントおよびキャンバスビュー
├── hooks/            # カスタムフック (エディタロジック、永続化)
├── data/             # 家具テンプレートおよび設定情報
├── utils/            # ロジック (キャンバス描画、間取り計算、PDF 生成)
└── types/            # TypeScript インターフェース定義
```

## セットアップ

1.  **依存関係のインストール:**
    ```bash
    npm install
    ```

2.  **開発サーバーの起動:**
    ```bash
    npm run dev
    ```

3.  **本番用ビルド:**
    ```bash
    npm run build
    ```
    - ビルドされた JavaScript および CSS アセットは `dist/assets` ディレクトリに出力されます。

## 開発ガイドライン

- **アーキテクチャの遵守:** 新しい機能はカスタムフックを利用したアプローチに従ってください。
- **レンダリング:** 重い描画ロジックを `useEffect` 内に含めないでください。描画ロジックは別のユーティリティファイル（例: `utils/isometricRenderer.ts`）に抽出してください。
- **状態管理:** Undo/Redo やデータ永続化などの複雑な状態管理は、カスタムフックを使用してカプセル化してください。
